# QR Code Generator

A single-page QR code generator that runs entirely in the browser.

## What it does

- Generates QR codes from a URL or text
- Lets you choose QR fill and background colors
- Supports transparent backgrounds
- Adds optional text in the middle of the QR code
- Downloads the finished QR code as a PNG

## Files

- `index.html` - the full app UI, styles, and JavaScript
- `qrcode.min.js` - the local QR generation library

## Run locally

Open `index.html` in a browser.

No build step is required.
